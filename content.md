To include normal text in a Markdown document, simply type the text as you would in any other text editor. Markdown will render it as plain text without any special formatting. Paragraph breaks can be inserted with a blank line. For example:

```markdown
This is a simple paragraph of text in Markdown. It will be displayed as normal text when rendered.

This is a second paragraph, separated by a blank line. Markdown recognizes the blank line as a paragraph break.
```
